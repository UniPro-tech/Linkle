# [日本語] セキュリティ ポリシー

## 脆弱性の報告

もしLinkleのセキュリティ的な脆弱性を発見した場合、以下のいずれかの方法でお知らせください。

### 書くべき項目

書けないところは空欄で構いません。

- 名前
- 連絡可能なメールアドレスもしくは電話番号
- どこで発見したか
- どこで発生しているか
- 再現する手順

### メール

[linkle@uniproject.jp](mailto:linkle@uniproject.jp)にメールをお送りください。

学園生は、Slackで あかつきゆいと(Yutaka Tani) 宛にDMを送信してください。
